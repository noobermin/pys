pys
===

Miscellaneous pythons.

Install
-------

Using `pip`, and installing locally,

    git pull https://github.com/noobermin/pys.git
    cd pys
    pip install . --user

Documentation
-------------

    >>> from pys import herp
    >>> help(herp)

or, from the `pys` directory,

    cd pys
    cat __init__.py

and repeat for other source files.
